# Документация по проекту «Портал АУСН»

## О проекте
Портал, посвящённый автоматизированной упрощённой системе налогообложения (АУСН), предназначен для малого бизнеса, индивидуальных предпринимателей и бухгалтерских специалистов. Цель проекта — упростить понимание режима АУСН, дать инструменты для расчёта налогов, предоставить понятные инструкции и привлечь клиентов на бухгалтерское обслуживание.

## Требования и ограничения
- **Простота и доступность**: понятные описания, смешанный экспертно-простой тон.
- **Открытый функционал**: без личного кабинета.
- **Адаптивность**: оптимальный UX для мобильных и десктопных пользователей.
- **Интеграция с amoCRM**: для приёма и обработки заявок через REST API.
- **Инструменты**: быстрый навигатор, калькуляторы, чек-листы, генераторы документов.
- **SEO**: поддержка метатегов и семантики.
- **Документирование**: вся история изменений и решений фиксируется в документации.

- Контакты

## Пользовательские сценарии
- Пользователь изучает АУСН и свои права по чек-листу.
- Использует калькулятор для расчёта налогов.
- Проходит интерактивный чек-лист соответствия.
- Скачивает пакет документов для перехода на АУСН.
- Оставляет заявку на консультацию, получает бесплатное предложение при заказе бухгалтерии.
- Изучает реальные кейсы и статьи в блоге.

## Технические решения
- **Frontend**: React (Cursor.dev)
- **Дизайн**: TailwindCSS
- **Ключевые компоненты**: калькулятор, чек-лист, формы заявок, блог (MDX или headless CMS)
- **Интеграция**: amoCRM (REST API) для приёма заявок
- **Документы**: скачивание и генерация для пользователей
- **Контент-компоненты**: отзывы и FAQ

### Проект: Портал по автоматизированной упрощённой системе налогообложения (АУСН)

#### О проекте
Портал призван стать навигатором для малого бизнеса по вопросам АУСН. Главная задача — упрощать переход и применение нового налогового режима, обучать, помогать и консультировать предпринимателей.

#### Требования и ограничения
- Фокус на максимально понятный смешанный тон.
- Функционал открыт — без личного кабинета.
- Поддержка мобильных устройств.
- Интеграция формы заявок с amoCRM.
- Калькуляторы, чек-листы и генераторы документов.

#### Структура сайта
Главная; Что такое АУСН; Как перейти на АУСН; Налоги и взносы; Новости и изменения; Инструменты и сервисы; Консультации и услуги; Блог и кейсы; Контакты.

#### Пользовательские сценарии
- Знакомство с режимом.
- Прохождение чек-листа прав на АУСН.
- Использование калькулятора налогов.
- Получение/генерация документов.
- Оставление заявки на консультацию.
- Ознакомление с кейсами блога.

#### Техническая реализация
- React (Cursor.dev), TailwindCSS
- Блог на MDX или headless CMS (Sanity/Strapi)
- Формы, обработка заявок в amoCRM (REST API)

- Документирование всех этапов разработки и изменений
